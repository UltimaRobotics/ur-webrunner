Available Topics

openwrt/metrics
Real-time system metrics updated every 5 seconds
openwrt/network/status
Network status changes (online/offline)
openwrt/services/+/status
Service status changes
openwrt/firmware/updates
Firmware update notifications

Control Topics (Publish To)

openwrt/services/+/control
Control services (payload: "start", "stop", "restart")
openwrt/system/reboot
Reboot the system (payload: "1")
